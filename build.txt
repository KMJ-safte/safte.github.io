
### 이미지 추가 방법

이미지는 세 가지 방법으로 추가할 수 있습니다[4]:

1. **웹 이미지**: `http://` 또는 `https://`로 시작하는 URL
2. **공용 디렉토리**: `/`로 시작하는 경로 (public 폴더의 이미지)
3. **상대 경로**: 마크다운 파일을 기준으로 한 상대 경로

## 자동 배포 프로세스

### Git을 통한 자동 배포

Netlify와 GitHub가 연결되어 있다면, 새 글을 작성한 후 다음 과정을 통해 자동으로 배포됩니다:

```bash
# 1. 변경사항을 Git에 추가
git add .

# 2. 커밋 생성
git commit -m "새 블로그 포스트 추가: 포스트 제목"

# 3. GitHub에 푸시
git push origin main
